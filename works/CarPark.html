Innovative Project


Edge Computing Car Park

The goal of this semester's project is to create an application listing the parking spaces available on the INSA campus. We are going to use cameras to film INSA parking spaces. Using a recognition algorithm, we will determine the free and occupied spaces and then make the data available on a server. On the user side, an application will be available and will, once the application has been launched, have a map showing the available places.

Analyse a real-life problem (4)

During this project, we had to analyze a problem, the loss of time to look for a parking space, and propose a solution to solve this problem.

Suggest a technological solution to a problem (4)

We have proposed a solution to the problem presented to us. By filming, using cameras, the parking spaces and analyzing the flow, we will determine the free spaces. Following this, we will communicate the data extracted from the analysis to a server which will take care of answering the requests of the phones.

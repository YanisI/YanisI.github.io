Data Processing and Analysis: Big Data


To familiarize ourselves with the R language, we first learned how to create and manipulate vectors in several ways. We gradually increased the difficulty by using new R-specific features like which or strsplit. Through several exercises, we were introduced to functions and finally to data sets. We have learned to manipulate data sets, in particular with the use of ddply which we have used a lot. Finally, we learned how to use ggplot to display data as a graph.

Subsequently, we had a data analysis project to carry out. We were asked to study a data set that we can choose. The analysis of this one had to bring out at least 5 graphs. We have chosen a data set on video game sales. It contained data on the sales of video games released between 1980 and 2016. Each game had data on its sales by region (Europe, Japan, North America, Others and total), its genre and its console.
